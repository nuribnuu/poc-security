🔥 Tel: Spoofing via Split View (POC)

Klik kanan pada tautan di bawah, lalu pilih "Open link in split view" (Chrome Canary dengan flag #side-by-side diaktifkan).

📞 Klik Kanan → Open link in split view (buka Google di split)

Setelah split view terbuka, tunggu 4 detik... akan muncul dialog tel: dari halaman ini (attacker) di atas halaman Google.

📌 Catatan: Kerentanan ini sudah diperbaiki di Chrome M141+ dengan menyembunyikan permission prompt dari tab tidak aktif.
Tapi di Safari 18.4 mungkin masih rentan. Uji coba pada Safari dengan langkah serupa (buka link di split view / tab baru lalu trigger tel: dari tab awal).
